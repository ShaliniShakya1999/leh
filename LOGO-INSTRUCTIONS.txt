LOGO SETUP - Infinity Routes Travels
=====================================

Your logo (infinity symbol with prayer flags + "INFINITY ROUTES TRAVELS" text) 
needs to be in this folder so it shows on the website.

DO THIS:
--------
1. Save your logo image (the one with prayer flags and red/blue text) 
   as:  logo1.png

2. Put logo1.png in THIS folder - the same folder where index.html is.
   Full path: D:\project\peyug_project\f6it_pro\leh\logo1.png

3. Refresh the website. The logo will appear in:
   - Header (top nav)
   - Hero section (home page)
   - Footer (all pages)

Format: PNG preferred (with transparent or dark background both work).
If your file is JPG, rename/save as logo1.png or we can change the code to logo1.jpg.
